# AIRE Services — Project Overview

## Goal
Build AIRE Services as a standalone product with:
- a modern public-facing website
- an admin/ops side for staff and management
- kiosk + mobile time tracking
- attendance visibility
- payroll-style reporting/export

## Stack recommendation
- Rails API backend
- React frontend
- Clerk auth (if retained after extraction review)

## Core sections
### Public
- Home
- Programs
- Team
- Discovery Flight
- Contact
- Careers (optional)

### Admin/Ops
- Dashboard
- Time Tracking
- Users
- Schedule
- Reports
- Kiosk
